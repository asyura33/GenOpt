GenOpt Examples for EnergyPlus

This directory contains example files that illustrate the use of GenOpt in conjunction with the EnergyPlus simulation program.

Because the Windows version of EnergyPlus writes its output files to the folder C:\Program Files\EnergyPlus\ExampleFiles\Outputs\, it was necessary to change the file RunEPlus.bat that is part of the EnergyPlus distribution. The changes allow GenOpt to run multiple EnergyPlus simulations in parallel. The modified file, called RunEPlusParallel.bat ensures that each simulation writes its output files to its own working directory, which is generated by GenOpt for each simulation. Typically, users do not have to modify any files because the GenOpt simulation configuration file (cfg\EnergyPlusWinXP.cfg) will cause GenOpt to start RunEPlusParallel.bat instead of RunEPlus.bat.

If you make a new directory with GenOpt and EnergyPlus input files, you will need to copy RunEPlusParallel.bat to this directory so it can be invoked by GenOpt.

For further information, please consult the GenOpt manual and the EnergyPlus manual.
