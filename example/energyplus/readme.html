GenOpt Examples for EnergyPlus

This directory contains example files that illustrate the use of GenOpt in conjunction with the EnergyPlus simulation program.

Information for EnergyPlus, versions 7.0.0 or later

Due to changes in the EnergyPlus *.eso file format, it is now required to specify in the optimization initialization file, section ObjectiveFunctionLocation the entries FirstCharacterAti (i=1, 2, ...). For example, a specification may now look like 

 Name1             = Q_heat;
 Delimiter1       = "170,";
 FirstCharacterAt1 = 1;
 
 Name2             = Q_cool;
 Delimiter2        = "201,";
 FirstCharacterAt2 = 1;
Without the keyword FirstCharacterAti, the wrong numbers may be read from the EnergyPlus *.eso file.

See the GenOpt manual for a description of this keyword.

Information for EnergyPlus on Windows, versions 5.0.1 or later

Because the Windows PATH variable does not contain the installation directory of EnergyPlus, the path to the EnergyPlus directory is hardcoded in the file "genopt_installation_dir"\cfg\EnergyPlus-*-Win*.cfg. Depending on your EnergyPlus installation, you may need to update in the file "genopt_installation_dir"\cfg\EnergyPlus-*-Win*.cfg the entry C:\\Program Files \(x86\)\\EnergyPlusV7-0-0\\RunEPlus.bat to point to the EnergyPlus directory that contains the file RunEPlus.bat.

Information for EnergyPlus on Windows, versions prior to 5.0.1

Because the Windows version of EnergyPlus writes its output files to the folder C:\Program Files\EnergyPlus\ExampleFiles\Outputs\, it was necessary to change the file RunEPlus.bat that is part of the EnergyPlus distribution. The changes allow GenOpt to run multiple EnergyPlus simulations in parallel. The modified file, called RunEPlusParallel.bat ensures that each simulation writes its output files to its own working directory, which is generated by GenOpt for each simulation. Typically, users do not have to modify any files because the GenOpt simulation configuration file (cfg\EnergyPlusWinXP.cfg) will cause GenOpt to start RunEPlusParallel.bat instead of RunEPlus.bat.

If you make a new directory with GenOpt and EnergyPlus input files, you will need to copy RunEPlusParallel.bat to this directory so it can be invoked by GenOpt.

For further information, please consult the GenOpt manual and the EnergyPlus manual.
